# 异步编程

# 是什么？
``` md
# 异步编程是什么？
  1. 异步编程就是让你的程序可以在**执行一个可能长期运行的任务同时**继续**对其他事件做出反应而不必等待任务完成**。与此同时，你的程序也将在任务完成后显示结果。
```

# 为什么？
``` md
# 为什么需要使用异步编程？
  1. 因为异步编程技术使你的程序可以在执行一个可能长期运行的任务的同时继续对其他事件做出反应而不必等待任务完成。与此同时，你的程序也将在任务完成后显示结果。
```
