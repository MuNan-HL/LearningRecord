# 注释规范
``` md
# 注释原则
  1. 要么不写, 要写就要写完整且后续需要不断维护.
```
``` md
# [HTML_注释规范](https://www.w3cschool.cn/wematy/wematy-n6s53bs7.html)
  1. 单行注释
  <!-- description -->

  2. 多行注释(和单行注释一样)
  <!-- description
  description -->

  3. 模块注释
  <!-- S description -->
  code
  <!-- E description -->

  4. 嵌套模块注释
  <!-- /description -->
```
-  [HTML_注释规范](https://www.w3cschool.cn/wematy/wematy-n6s53bs7.html)

``` md
# [CSS_注释规范](https://www.w3cschool.cn/wematy/wematy-qaru3bsh.html)
  1. 单行注释
  /* description */

  2. 多行注释(和单行注释一样)
  /* 
   description
   description
   */

  3. 模块注释
  /* Module A
  ---------------------------------------------------------------- */

  4. 文件信息注释
  /* 
   * @description
   * @
   */
```
-  [CSS_注释规范](https://www.w3cschool.cn/wematy/wematy-qaru3bsh.html)
