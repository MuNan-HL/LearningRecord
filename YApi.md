# YApi 可视化的接口管理平台
``` md
# YApi 是什么？
  1. YApi 是一个可本地部署的、打通前后端及QA的、**可视化的接口管理平台**
```

``` md
# 为什么使用 YApi？
  1. 提升接口文档的维护准确度和效率，通过idea的easyYapi插件自动将controller的接口生成为接口文档

  2. 提升自测和联调效率
```

``` md
# 怎么使用 YApi？
  1. 进行 YApi 本地部署

  2. 生成接口文档
> 使用 EasyYAPI 插件

  3. 自测接口
> 使用 cross-request 插件
```
