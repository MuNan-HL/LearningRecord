# 前端开发
-
``` md
# 前端开发是什么?
> 1. 是前后端分离的产物，前端负责可交互页面的开发。负责开发**静态可交互页面和的动态可交互页面**, 通过_HTML, CSS, JS_以及_Vue, elementUI, VSCode_等**前端开发工具**去开发**静态可交互页面和动态可交互页面**.
```

-
``` md
# 前端开发的分类
> 1. CMS 内容管理系统的前端开发

> 2. 门户网站的前端开发
```

-
``` md
# 为什么需要前端开发?
> 1. 在前后端分离的背景下, 导致了前端开发主要负责**可交互页面**的开发，后端开发主要负责**数据**开发.
```
