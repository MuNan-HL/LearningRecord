# 代码调试技巧
``` md
# [JS 调试技巧](https://zh.javascript.info/debugging-chrome)
  1. 使用 **debugger** 打断点

  2. **下一步**(执行下一行代码, 遇到函数会进入函数内部执行. 与**步入**的区别在于遇到异步函数时的处理机制不同, 它会**忽略异步行为**)

  3. **跨步**(执行下一行代码, 遇到函数不会进入函数内部执行)

  4. **步入**(执行下一行代码, 遇到函数会进入函数内部执行)

  5. **步出**(直接执行到当前函数的末尾)
```
